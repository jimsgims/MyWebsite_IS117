Becoming Anonymous

If you ever wanted to become anonymous, this guide goes over some of the easiest and most extreme ways an individual can go private on the Internet. For a frame of reference, the average person should at the very least be following the basic measures, with the more concerned person taking the intermediate measures. The advanced and extreme measures are for people whom are passionate about online privacy and want to keep themselves practically off the grid.

Basic Measures

  • Use a password manager and keep all your passwords you use for all the website you visit inside of it. Your passwords need to be strong; strong passwords have a lot of numbers, letters, and symbols. A password manager will generate passwords for you, but you will still need to remember a master password for your password manager.
  • 2-Factor Authentication. This is an additional security measure after you input your password for a website. This is typically in the form of a text message, but lately companies are transitioning to an app called Google Authenticator. The latter is more secure, but either methods will help you secure your accounts.
  • The web browser you use. For a basic measure, you can use whatever browser you want, be it Microsoft Edge, Google Chrome, Opera, or any other web browser. If you install any addons to your browser though, make sure it is a trusted source. Finally, you will want to switch your search engine (the websites like Google or Yahoo!) to something more secure like duckduckgo or startpage
