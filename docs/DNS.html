Domain Name System

How does visiting a website work?

Most people never see or care about IP addresses, but instead utilize the Domain Name System (DNS) to associate web addresses with IP addresses. In a short explanation, computers ask DNS servers for IP addresses whenever you visit a website. DNS servers will search through computer networks worldwide to find the website's associated IP address. Once that's done, the DNS server will send the IP address back to the computer that requested it.

A graphical example of how the DNS works.

More about the DNS

DNS servers are connected through a distributed hierarchy. In a sense, each domain is divided into zones. A domain is what comes after the last dot of a website. Some domain examples include .org, .com, and .net.

The design of the DNS makes it an open and public communication protocol, designed to scale no matter how much the Internet grows. Its open architecture makes it susceptible to cyber attacks through a method called DNS spoofing. This is when a hacker modifies a DNS server and changes the IP addresses of domain names, essentially sending people to an imposter website under a trusted web address.
